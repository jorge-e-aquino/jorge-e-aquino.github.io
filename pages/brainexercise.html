WORK RESUME ABOUT

The Brain Exercise Initiative is a non-profit 501(c)(3) that "uses simple math, writing and reading aloud exercises as an intervention to improve cognitive function in those with Alzheimer's Disease" and has 35 chapters across the United States. The organization was founded in UCLA and is now in four countries.

My contributions are visible on the GitHub repository. I worked with React Native Elements, a library of iOS/Android friendly components and an implementation of the Material Design System. I became familiar with React Native concepts such as the StackNavigator, PropTypes, and the View.

My primary task besides giving presentations was designing custom components.
